Project Dataset

By Monica Meyer

SF Mobile Food Schedule

I chose this dataset because I wanted the opportunity to visualize data on a map. I thought that this mobile food truck data would be interesting and useful because it also includes the days and times of the permit, thus indicating what times the food truck will be located at any given latitude/longitude position. Hopefully I can create an interactive visualization that allows a user to see what trucks are nearby at specific times. If I have trouble with how large the dataset is, I may subset it to weekdays and/or one neighborhood of San Francisco.

The dataset can be found here on the SF OpenData page, or the subset currently being used for my visualization can be found on my github repository. The data is free for use by the public.

The dataset contains both numerical and categorical features. The categorical features include day of the week, start and end times, permit location and applicant (restaurant or company name). The numerical features include both X, Y and latitude/longitude coordinates of the food truck. There is also an optional text field containing more information the type of food being served. Currently, the only modifications of the dataset I have performed are to remove several columns that will not be used for the project.
