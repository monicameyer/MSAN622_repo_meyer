Visualization Prototype

Dataset Summary

This dataset is part of the San Francisco Climate and Health Profile, a report identifying locations most vulnerable to climate change health outcomes. The dataset includes 40 different variables used to provide quantitative measurements of a given neighborhood's resiliency and vulnerability to climate change stressors as well as several aggregated scores indicating overall resiliency by different standards (environmental, economic, hazard, housing, etc). For more information, view the project dataset page.

Sketch Summary

My plan is to implement a map of San Francisco that will allow filtering so the user can pick between several columns of the data to see on the map. I will have a bubble chart indicating several of the hazard risk columns of the dataset, and a small multiples bar chart showing some health related columns (shelter rate, previous hospitalizations, population density, etc). I have added a parallel coordinates plot because it is the best way to visualize each of the associated scores (all on scales of 1 to 5) of the data. For more information, view the project sketch page.
